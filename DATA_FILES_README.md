# 数据文件说明

## 数据文件

由于GitHub对文件大小的限制（100MB），以下数据文件不会被提交到版本控制中：

- `hand_gesture_data.npz` - 原始收集的手势数据
- `hand_gesture_data_augmented.npz` - 数据增强后的手势数据

## 如何重新生成数据

### 1. 收集原始数据
```bash
python collect_real_time.py
```
- 按 0-9 选择手势
- 按空格开始录制
- 每个手势收集 50-100 个样本

### 2. 数据增强
```bash
python augment_data.py
```
- 将原始数据扩展到4000个样本
- 应用各种数据增强技术

### 3. 训练模型
```bash
python train_improved.py
```
- 使用交叉验证训练模型
- 保存最佳模型为 `gesture_gru_cv.pth`

## 文件大小说明

- 原始数据 (~10MB): 100个手势 × 30帧 × 225特征
- 增强数据 (~200MB): 4000个手势 × 30帧 × 225特征

## 注意事项

- 数据文件包含个人隐私信息，请妥善保管
- 如需分享数据，请使用压缩包或云存储
- 模型文件 (.pth) 已包含在版本控制中
